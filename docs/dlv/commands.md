# Delve 命令清单

本文档列出 Delve 调试器的所有可用命令，按功能分类。

## 程序执行控制

| 命令 | 别名 | 描述 |
|------|------|------|
| `call` | - | 恢复进程并注入函数调用（实验性功能） |
| `continue` | `c` | 运行直到断点或程序终止 |
| `next` | `n` | 单步执行，跳过函数调用 |
| `next-instruction` | - | 单步执行一条 CPU 指令，跳过函数调用 |
| `rebuild` | - | 重新构建目标可执行文件并重启 |
| `restart` | - | 重启进程 |
| `rev` | - | 反向执行指定命令 |
| `rewind` | - | 反向运行直到断点或历史记录起点 |
| `step` | `s` | 单步进入函数 |
| `step-instruction` | - | 单步执行一条 CPU 指令 |
| `stepout` | `so` | 执行直到当前函数返回 |

## 断点管理

| 命令 | 描述 |
|------|------|
| `break` | 设置断点 |
| `breakpoints` | 打印活动断点信息 |
| `clear` | 删除断点 |
| `clearall` | 删除多个断点 |
| `condition` | 设置断点条件 |
| `on` | 断点命中时执行命令 |
| `toggle` | 切换断点开关状态 |
| `trace` | 设置跟踪点 |
| `watch` | 设置观察点 |

## 变量和内存检查

| 命令 | 描述 |
|------|------|
| `args` | 打印函数参数 |
| `display` | 每次程序停止时打印表达式值 |
| `examinemem` | 检查指定地址的原始内存 |
| `locals` | 打印局部变量 |
| `print` | 计算并打印表达式 |
| `regs` | 打印 CPU 寄存器内容 |
| `set` | 修改变量值 |
| `vars` | 打印包变量 |
| `whatis` | 打印表达式类型 |

## 线程和协程管理

| 命令 | 别名 | 描述 |
|------|------|------|
| `goroutine` | `gr` | 显示或切换当前协程 |
| `goroutines` | `grs` | 列出程序协程 |
| `thread` | `tr` | 切换到指定线程 |
| `threads` | - | 打印所有跟踪线程信息 |

## 调用栈操作

| 命令 | 描述 |
|------|------|
| `deferred` | 在延迟调用上下文中执行命令 |
| `down` | 向下移动当前栈帧 |
| `frame` | 设置当前栈帧或在指定帧上执行命令 |
| `stack` | 打印栈跟踪 |
| `up` | 向上移动当前栈帧 |

## 其他命令

| 命令 | 描述 |
|------|------|
| `check` | 在当前位置创建检查点 |
| `checkpoints` | 打印现有检查点信息 |
| `clear-checkpoint` | 删除检查点 |
| `config` | 更改配置参数 |
| `disassemble` | 反汇编代码 |
| `dump` | 从当前进程状态创建核心转储 |
| `edit` | 在编辑器中打开当前位置 |
| `exit` | 退出调试器 |
| `funcs` | 打印函数列表 |
| `help` | 打印帮助信息 |
| `libraries` | 列出已加载的动态库 |
| `list` | 显示源代码 |
| `packages` | 打印包列表 |
| `source` | 执行包含 Delve 命令的文件 |
| `sources` | 打印源文件列表 |
| `target` | 管理子进程调试 |
| `transcript` | 将命令输出追加到文件 |
| `types` | 打印类型列表 |
