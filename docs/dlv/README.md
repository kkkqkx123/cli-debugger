# Delve (dlv) 调试器文档

## 概述

Delve 是 Go 编程语言的调试器，设计简单、功能完整且非侵入式。它提供了丰富的命令行接口，支持断点管理、变量检查、执行控制、线程和协程管理等功能。

## 启动方式

```bash
# 调试主程序
dlv debug [package]

# 附加到运行中的进程
dlv attach <pid>

# 执行预编译的二进制文件
dlv exec <binary>

# 连接到远程调试服务器
dlv connect <addr>
```

## 命令分类

Delve 的命令按功能分为以下几大类：

1. **程序执行控制** - 运行、暂停、单步执行
2. **断点管理** - 设置、删除、条件断点
3. **变量和内存检查** - 查看变量、内存、寄存器
4. **线程和协程管理** - 列出和切换线程/协程
5. **调用栈操作** - 查看栈、切换栈帧
6. **其他工具命令** - 配置、反汇编、检查点等
